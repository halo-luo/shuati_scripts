# shuati_scripts
刷题脚本，加载题库刷题
该Python程序是一个**题库学习工具**，主要功能包括：
1. **加载题库**：支持Excel格式的题库文件（需包含"题干"、"选项"、"答案"等列）
2. **交互式答题**：逐题展示题目、选项，支持多选操作
3. **智能判答**：实时判断答案正误，高亮显示正确/错误选项
4. **错题管理**：自动记录错题到本地数据库
5. **数据导出**：可将错题本导出为Excel文件
6. **进度控制**：支持前后导航、跳转指定题号

### 使用说明
#### 1. 运行环境要求
```bash
pip install pandas openpyxl
```

#### 2. 启动程序
```python
if __name__ == "__main__":
    root = tk.Tk()
    app = QuizViewer(root)
    root.mainloop()
```

#### 3. 核心操作流程
| 步骤 | 操作说明 | 界面元素 |
|------|----------|----------|
| ① 加载题库 | 点击"选择Excel题库"按钮，上传符合格式的题库文件 | 顶部工具栏 |
| ② 开始答题 | 系统自动展示第一题，勾选选项后点击"确认答案" | 中央答题区 |
| ③ 查看结果 | 系统显示答案解析和法律依据 | 绿色/红色结果提示 |
| ④ 错题管理 | 自动收录错题到错题本，可随时查看/导出 | 底部状态栏 |

#### 4. Excel题库格式要求
必需列名：
- `题干` (题目内容)
- `选项` (用`|`分隔的选项，如："A.选项1|B.选项2")
- `答案` (如："AB"表示多选)
- `题目依据` (可选，法律条文依据)

示例：
```excel
序号 | 题型 | 题干                     | 选项                 | 答案 | 题目依据
1   | 单选 | 《宪法》规定...          | A.全国人大|B.国务院 | A    | 宪法第XX条

```

#### 5. 特色功能
- **智能判卷**：支持多选题部分正确识别
- **错题统计**：实时更新错题数量及详情
- **进度记忆**：保持当前答题进度
- **响应式布局**：自适应不同屏幕尺寸

> 注意：程序会自动处理Excel文件中的空格和格式问题，但建议保持规范的数据格式以获得最佳体验。
